Those white boxes are actually buttons. Yeah, I know.

The first and last points of a curve are the bezier control points.

The whole thing is greyscale because I'm cheap.

Keys
----

Delete: delete
Up: Move shape up (z-axis)
Down: Move shape down (z-axis)
s: set stroke
w: set strokeweight
f: set fill
c: Change shape mode (curve/polygon)
m: Change mode (select/draw)
